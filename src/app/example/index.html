这是example页面

1. 开发服务器启动时默认打开example页面，您可以通过修改<project>/config/dev.config.js文件中的'home'属性来改变默认打开页面。或者运行命令 

sbr run [pageName] //eg: sbr run example/normal

2. 更多的命令可以通用运行 'sbr -h 查看'.

3.下面是一些有用的examples.

  • 默认页面

    webpack自动引入index.js
    同时index.js自动引入index.less

  • 传统的开发方式

    使用一些传统的开发方式来进行工程开发

  • 使用代理服务器

    使用代理服务器进行开发
    配置 [projectName]/config/dev.config.js 中的proxyTables来进行代理服务器配置

  • 模板化开发

    传统的模板化开发方式
    使用underscore模板

  • 使用精灵图

    在png图片的后面加上?sprite的图片请求会自动打包到精灵图里

  • 使用vue

    saber-cli已经为您默认安装一些常用的vue依赖包
    配置 [projectName]/config/dev.config.js 中的vendors来减少你项目中的文件大小
